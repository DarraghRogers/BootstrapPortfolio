Number Facts Generator

In this project, I wanted to learn about API's and how to use one. This is simply a fact generator using an API from http://numbersapi.com/. You enter a number into the box provided and a fact will appear to do with that particular number. You also have the option to choose facts about dates instead of numbers. It uses Ajax to retrieve the informarion from the API. I have also used a bootstrap helper for the styling.
